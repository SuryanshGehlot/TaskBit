## üöÄ About The Project

TaskBit is a task management web application built with React.js featuring:
- Drag-and-drop and arrow-button reordering supporting 100+ tasks smoothly.
- Real-time filters and counters for task states: All, Active, Completed.
- Task completion animations for enhanced user interaction.
- Clean, responsive UI with custom CSS supporting multiple browsers.
- Inline task editing via an easy-to-use Edit button or double-click.

This modern, lightweight app helps users organize tasks efficiently with intuitive controls.

## üîß Getting Started

### Prerequisites

- Node.js (>=14.x recommended)
- npm (comes with Node.js)

### Installation


1. Clone the repo:  git clone https://github.com/SuryanshGehlot/TaskBit.git

2. Navigate to project folder:  cd TaskBit

3. Install dependencies:  npm install

4. Run the development server:  npm run dev

5. Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.

---

## üéØ Usage

- Add a new task with the **Add Task** button.
- Reorder tasks via drag-and-drop or arrow buttons.
- Toggle task completion with the checkbox.
- Edit task title by clicking **Edit** or double-clicking the task.
- Filter tasks by All, Active, or Completed states in real-time.
